
So here the wordlist should be according to the parameter, if its id it should be numbers, user we can find or create a word list and so on.

Creatin a number wordlist for ids
```shell-session
for i in $(seq 1 1000); do echo $i >> ids.txt; done
```

```bash
ffuf -w ids.txt:FUZZ -u http://admin.academy.htb:31976/admin/admin.php -X POST -d 'id=FUZZ' -H 'Content-Type: application/x-www-form-urlencoded' -fs 768
```
