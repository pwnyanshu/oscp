# Hashcat

A GPU-accelerated password-cracking tool for Linux, Windows, macOS.

---

## Basic Syntax

```bash
hashcat -a <attack_mode> -m <hash_type> <hashes> [wordlist/rule/mask/...]

hashcat -a 0 -m 0 <hash> /usr/share/wordlists/rockyou.txt

hashcat -a 0 -m 0 <hash> rockyou.txt -r /usr/share/hashcat/rules/best64.rule

hashcat -a 3 -m 0 <hash> '?u?l?l?l?l?d?s'

-a 0: Dictionary attack
-a 3: Mask attack - A structured brute force pattern
-m 0: MD5
-m 100: SHA1
-m 1400: SHA256
<hashes>: single hash or file of hashes
Last argument: depends on attack mode (wordlist, rules, masks, etc.)
-r : using rules
```

## Rules (Wordlist Mutations)

- Rules modify each word (append numbers, leetspeak, uppercase, reverse, etc.)
- Rule files found in `/usr/share/hashcat/rules/`
- Example (using best64 rules):

```
hashcat -a 0 -m 0 <hash> rockyou.txt -r /usr/share/hashcat/rules/best64.rule
```

## Mask Attack (Mode `-a 3`)

- User defines a structured brute-force pattern.

|Symbol|Charset|
|---|---|
|?l|abcdefghijklmnopqrstuvwxyz|
|?u|ABCDEFGHIJKLMNOPQRSTUVWXYZ|
|?d|0123456789|
|?h|0123456789abcdef|
|?H|0123456789ABCDEF|
|?s|«space»!"#$%&'()*+,-./:;<=>?@[]^_`{|
|?a|?l?u?d?s|
|?b|0x00 - 0xff|
### Example Mask

Password format: Uppercase + 4 lowercase + digit + symbol  
Mask:

```
?u?l?l?l?l?d?s
```

Command:

```bash
hashcat -a 3 -m 0 <hash> '?u?l?l?l?l?d?s'
```

## Practical Workflow

1. Identify hash type (`hashid`, `hashcat --help`)
2. Start with dictionary attack
3. Add rules if dictionary fails
4. Use masks when structure of password is known

---

## Reminder

Hashcat only cracks **weak** passwords. Strong, long, random passwords are computationally painful even with GPUs.
