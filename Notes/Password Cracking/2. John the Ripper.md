
A fast, rule-based password-cracking tool for UNIX/Linux/Windows hashes.  
Best version: **Jumbo JtR** → more wordlists, more formats, better optimizations.

---
## Cracking modes

#### Single crack mode

Super useful for Linux passwd+shadow.
Uses the victim’s metadata to generate likely passwords.
We **never manually give** GECOS fields to John.
John extracts them **automatically from the passwd file** _itself_.

```bash
john --single passwd
```


#### Incremental mode
If everything fails, this mode tries _every combination_, but smartly ordered.

```bash
john --incremental <hash_file>

# Incremental configs live in `/etc/john/john.conf`.
```

## Cracking files

```shell-session
<tool> <file_to_crack> > file.hash
```
