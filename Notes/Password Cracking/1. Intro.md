
Ways to do it

### Rainbow tables
Already precompiled list of passwords, but fails (if the website has salted the passwords before storing)


### Bruteforce attack
Fast for MD5 but slow for latest hashing algos.


### Dictionary attack
Wordlist attack great, JohnTheRipper, hashcat
Major wordlists -  [rockyou.txt](https://github.com/brannondorsey/naive-hashcat/releases/download/data/rockyou.txt) or [SecLists](https://github.com/danielmiessler/SecLists).



Hash a password in shell
```shell-session
bmdyy@htb:~$ echo -n Soccer06! | md5sum
bmdyy@htb:~$ echo -n Soccer06! | sha256sum

DONT FORGET -n
```
